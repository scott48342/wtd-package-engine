## ~8:52 PM - Tire Search Bug Fix

Scott reported tire search working for Ford trucks but not for 2005 Chrysler Town & Country.

**Root cause:** Model name slugification for Wheel-Size API was wrong.
- `"Town & Country"` was being converted to `"town-and-country"` (my first fix)
- But Wheel-Size API actually uses `"town-country"` (no "and")

**Fix:** Updated `/api/vehicles/tire-sizes/route.ts` to properly slugify model names:
```ts
// Before: "Town & Country" → "town-and-country" ❌
// After:  "Town & Country" → "town-country" ✓
const modelSlug = model.toLowerCase().replace(/[^a-z0-9]+/g, "-").replace(/^-+|-+$/g, "").replace(/-+/g, "-");
```

**Result:** Tire search now returns 6 OEM sizes for Chrysler Town & Country:
- Stock: 215/65R16, 215/60R17
- All: 215/65R16, 215/60R17, 235/55R17, 235/50R18, 215/70R15, 225/55R18
